#  Retail Sales Analysis

# Project Overview

This project presents an end-to-end Retail Sales Analysis using SQL, Excel, and Power BI. 
The objective is to analyze sales performance, customer behavior, product profitability, 
and regional performance to generate business insights and support data-driven decision-making.


##  Business Problem

Retail businesses generate large volumes of sales data every day. This project helps answer key 
business questions related to revenue, profit, customer purchasing behavior, product performance, 
and sales trends through data analysis and visualization.
